MassMover

Bot

Move All Users From a Discord Channel Fast !

How To Use:

  1. Invite the MassMover Bot to your discord server.
  2. If you are moving up to 10 users, add the LVL 1 POWER UPS
  3. If you are moving up to 20 users, add the LVL 1 and LVL 2 POWER UPS
  4. If you are moving more than 20 users, add the LVL 1, LVL 2 and LVL 3 POWER UPS
  5. Make sure the channels are not blocking "Move" permissions.
  6. Type "> help" in a channel the bot can read and write.
  7. Only users with Move permission will be able to use the bot !

    8. Done!
